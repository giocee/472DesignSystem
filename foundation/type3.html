Typography

Montserrat & Raleway

The typography hierarchy uses two sans-serif typesfaces.

Using two sans serif typefaces make this hierarchy appear clean and modern. This hierarchy might be better suited to pages with little text content.

This hierarchy uses the perfect fourth scale with a base font size of 18px.
