Design System

Giovanni Centeno Portfolio Website

About the Design System

Professional. Forward-Thinking. Human.

This website documents the overall style, brand, page layout, and major components featured in the Giovanni Centeno Portfolio website. This design system website is meant to be all encompassing for the Giovanni Centeno Design brand. Adherence to this design system's guidlines allows for maximum consistiency across the Giovanni Centeno brand.

This design system was created by Giovanni Centeno and uses the Bootstrap 4 framework.

Design System Organization

The design system is organized into three sections:

Foundations

Elements in the Foundations section document guidelines for the typography hierarchy, color palette, and logo usage. Using these elements are essential when creating media to represent the Giovanni Centeno brand.

Components

Components are essentially building blocks to use within the portfolio website. The design system uses three major components: Navigation, Buttons, and Cards. All components in this design system are shown with an example, accessibility guidlines, best usage practices, and code.

Page Patterns

Page patterns are the layouts of the pages featured on the website. In this section three page layouts are shown with multiple screen sizes with detailed measurements and CSS code snippets.
