The design system usestwo button styles. Buttons are used as another means of nagivation through the website and are repsonsive to screen size.

Accessibility

Ensure accessibility by using descriptive button labels for users using assistive technologies.

Also, all buttons should only direct to one page and clearly state the page they direct to.

Use

Use responsive buttons bar on all pages. To see specifics on placement, see page patterns

Best Practices

  • Ensure the buttons direct to a working existing page
  • Begin buttons with action oriented verbs Go to, Back to, or View the
  • Use Hover states with darker colors for all buttons. See Button Colors
