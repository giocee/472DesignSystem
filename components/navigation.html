Navigation

The Nagivation Bar is an essential component in the design system. The Nav bar is used for all nagivation through the website and is repsonsive to screen size.

Example

The example is responsive. Adjust the web browser window to demonstrate the responsive behaviors.

  • Home
  • Link
  • Link
  • Link

Accessibility

Ensure accessibility by using a <nav> element or, if using a more generic element such as a <div>, add a role="navigation" to every navbar to identify it as a landmark region for users using assistive technologies.

Also include ARIA tags: aria-expanded="[true or false]" and aria-current="page" to help assistive technologies recognize which pages are active and which menus are expanded.

Use

Use responsive navigation bar on all pages. To see specifics on placement, seesee page patterns

Best Practices

  • Ensure the logo image is a clickable that link directs users to the Home page
  • Begin links with Home page link
  • Order links from most important to least important
  • Differentiate current page with darker text color
  • Never use more than 5 top level navigation links
